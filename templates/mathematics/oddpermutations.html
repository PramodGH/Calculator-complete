How to Calculate Odd Permutations?

Below are some of the detailed steps on how to find Odd Permutations of a number "n". Go through them and solve concerned problems in no time.

  • The Initial Step in finding the Odd Permutations is to find the factorial of the given number n.
  • Do remember that n value is greater than or equal to 2 i.e. n>=2
  • Simply divide the factorial of number by 2 i.e. n!/2
  • The final value obtained itself is the Odd Permutation for the number n.

Example: Calculate the Odd Permutations for n = 5?

Solution:

Given n value = 5

Formula to find Odd Permutations = n!/2 for n>=2

Find the factorial of 5 as a part of the initial process

5! = 5*4*3*2*1

= 120

Substitute the value of 5! in the formula

Odd Permutations = 5!/2

= 120/2

=60

Therefore, the number of ways that can be arranged for n = 5 is 60.
