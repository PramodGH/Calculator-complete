How to Calculate Permutations Replacement?

In Permutations Replacement, a sample of r elements is taken from a set of n distinct objects, order matters, and replacements are allowed.

If we pick r elements from a set having a size of n then each element r can be chosen n ways. The entire sequence of r elements is called a string and can be chosen in n x n x n x n ..... x n = nr ways.

PR(n,r)=nr

Where p is the number of permutations.

n is the total number of elements in the set,

and r is the number of elements you choose from this set.

Plugin the values of n, r and solve the equation to get the possible ways.

Example: Choose a sequence of 2 Letters from an Alphabet Size of 4 Letters {a,b,c,d} where repetitions are allowed?

Solution:

From the given data n = 4 r = 2

The formula for Permutations Replacement or Repetition is PR(n,r)=nr

Substituting the values of n, r in the formula and we get the equation as follows

PR(4, 2) = 42 = 16

Therefore, there are 16 ways to choose a sequence of 2 letters from an Alphabet Size of 4 Letters {a,b,c,d}.
